02 GitHub Pages (Free Forever)

If you have a GitHub account, this gives you a permanent free URL!

  1. Create a new repository on GitHub
  2. Name it something like egg-timer
  3. Upload your files (index.html, index.css, app.js)
  4. Go to Settings ‚Üí Pages ‚Üí Select "main" branch ‚Üí Save
  5. Your URL will be: https://yourusername.github.io/egg-timer
CREATE GITHUB REPO ‚Üí
03 Vercel (Also Very Easy)

Another great free option with instant deployment!

  1. Go to vercel.com and sign up (free)
  2. Click "New Project"
  3. Import from GitHub or upload directly
  4. Get your URL: your-project.vercel.app
04 Update Your QR Code

After deploying, update the URL in your app so the QR code works:

  1. Open app.js in a text editor
  2. Find this line near the top:
const DEPLOYED_URL = 'https://egg-master-timer.netlify.app';

Change it to your actual deployed URL:

const DEPLOYED_URL = 'https://YOUR-ACTUAL-URL.netlify.app';

Save the file and re-upload if needed!
